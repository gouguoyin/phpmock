[PHPRAP，是一个PHP轻量级开源API接口文档管理系统，致力于减少前后端沟通成本，提高团队协作开发效率，打造PHP版的RAP。](http://phprap.gouguoyin.cn)

## 使用

 - 引入类
 
     ```php
      include_once($_SERVER['DOCUMENT_ROOT']."/mock.php");
     ```
     
 - 实例化类
 
     ```php
      $mock = new mock();
     ```
